# Next Causal Target Recommendation

## Recommendation

The next highest-information target is **schema realization / exact `tool_calls` envelope normalization on the frozen scaffold**.

## Why

The completed evidence now rules topology down to a weakened contributor:

- anchor concentration helps, but is not sufficient;
- topology changes the curve, but does not recover the H1/H2 floor;
- Phase U shows that a tiny schema patch can preserve safety while still failing on capability;
- Phase R already identified the outer-envelope failure as the strongest remaining hypothesis;
- H1 and H2 both succeeded inside a narrow, frozen control scaffold rather than via broad corpus redesign.

## Interpretation

The remaining unresolved question is not whether the model can be made to use tools at all.
That is already established by H1 and H2.

The unresolved question is how to restore the exact realization behavior:

- strict `tool_calls` envelope,
- correct outer schema,
- stable tool-family realization,
- and a prompt regime that does not wash out the envelope cue.

## What Not To Do Next

- Do not launch another topology branch.
- Do not launch another anchor-concentration sweep.
- Do not broaden the corpus before the envelope question is isolated.

## Next-Step Shape

The next experiment should be a control-preserving schema-realization study, with the frozen scaffold held fixed and the `tool_calls` envelope treated as the primary causal object.
