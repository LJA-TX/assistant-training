1. surface is structurally legitimate, but governed upstream evidence required for meaningful lifecycle continuation is absent on the current frozen corpus;

### Blocker-Oriented Branch

Exemplar surface:

1. `direct_answer_substitution_count`

Current branch meaning:

1. lifecycle entry is legitimate on the current corpus and ownership chain;
2. active blocker work is required before any later readiness or gate progression can become meaningful.

High-level blocker-oriented sequence:

1. blocker entry qualification;
2. blocker persistence validation;
3. blocker characterization;
4. preservation-versus-completeness attribution when needed;
5. blocker decision point.

## Why Adoption Is Conditional

Conditional status remains necessary because:

1. only one active blocker-oriented surface family has completed the sequence;
2. no blocker-oriented surface has yet traversed a post-blocker transition;
3. additional evidence may refine which blocker-oriented steps are always mandatory versus conditionally mandatory.

## Future Reassessment Expectations

Reassessment is expected after any of:

1. another blocker-oriented surface reaches characterization;
2. a blocker-oriented surface reaches an authorized post-blocker transition;
3. future evidence shows the current blocker-oriented sequence needs revision;
4. later regimen retrospectives revisit branch structure using broader surface diversity.

## Preservation Boundary

This carry-forward artifact does not:

1. authorize migration;
2. authorize scorer-pathway work;
3. authorize detector or threshold changes;
4. alter any existing surface state;
5. create new doctrine.
