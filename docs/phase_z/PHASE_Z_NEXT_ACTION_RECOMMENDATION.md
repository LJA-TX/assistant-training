# Phase Z Next Action Recommendation

## Determination

**2. Proceed next to Treatment A**

## Rationale

The control arm is a valid ablation result, but it does not reach or approach the H1 capability floor.

This is a promotion failure, not an attribution failure and not an operational failure:

- promotion failure: yes, because the candidate is not promotable;
- attribution failure: no, because the scaffold, dataset, and frozen evaluation contract stayed intact;
- operational failure: no, because preflight, training, and canonical evaluation all completed successfully.

## Why Treatment A Should Follow

The Phase Y / Phase X design is a low-to-high ablation sequence.

Control is the baseline arm; it establishes the lower anchor-concentration end of the causal sweep. The next information-maximizing step is therefore Treatment A, not stopping early.

Stopping now would leave the anchor-concentration curve under-sampled and would not answer whether the capability lift, if any, appears as anchor share increases.

## Interpretation Of The Safety Regression

The control arm’s adversarial no-call collapse is important, but it does not invalidate the ablation setup.

It does mean the control arm is not promotable and that later arms must be judged against the same frozen safety contract.

## Recommended Next Action

Proceed to Treatment A under the same governed framework.
